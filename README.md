#神杀国战LUA技能手册
LuaSkillsForQSGSHegemony
##简介
本手册适用于太阳神三国杀国战最新版本，手册共分为A~Z共26章，技能包括国战标准版、国战阵包、国战势包的所有武将技能(计划私更补完身份局技能)。所有技能按首字母分类到对应的章节文件中。  
为了方便读者查阅，本手册还有对应的Markdown文件，可以点击技能链接跳转到对应的技能代码处。
